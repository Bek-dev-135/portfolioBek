Password Checker

This python code fetches all compromised passwords from the well-known hackaton webpage, but since we don't want to put our actual password on there and search it with the risk of sending our password to them, what we could do is give a little portion of the hash version of our password, and when the website gives us a list of passwords that match those first few strings, we scrape that list to our device and try to match the rest of our passwords from what's on that list. Now we can rest assured that we didn't give our full password to an unknown website, and we can calculate from the data that it gives us based on the bits of password we offered. The code for this could be found here (link to github) file name passwordchecker.py

Power automate tool- Free calendar slots finder

I will share below one of many tools we have worked on to automate and save the time of the assistant service line. It dynamically adjusts to time zones and meeting times. The making of this tool was followed by a testing phase where I learned how feedback significantly helps eliminate issues for the end user. I hope that this tool will be used by a large number of people to streamline the process of scheduling meetings.
