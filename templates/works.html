Security Utility

Password Checker

This Python code is designed to check whether a password has been compromised by querying a well-known data breach database (such as the one used in hackathons or by Have I Been Pwned). To protect user privacy, the script does not send the full password to the website. Instead, it generates a hashed version of the password and only sends the first few characters of the hash. The website then returns a list of hashes that match that prefix. The script downloads this list and compares it locally with the full hash of the user's password.

This method ensures that the full password is never exposed to any external site, maintaining security while still allowing users to check if their credentials have been compromised.
